Two Modes

  • Text mode is the default mode, where everything appears as normal text. You should use this mode for anything that is not mathematical notation.
  • Math mode should be used when typing any mathematical notation, including variables, numbers, functions, sets, etc.

You can enter math mode in two ways:

  1. Enclose mathematical notation with dollar sign ($), which will create in line math environment. For example, $E=mc^2$ will display \(E=mc^2\)
  2. Enclose mathematical notation with double dollar sign \($$\), which will create a separate line of math environment. For example, \(\textrm{\$\$a^2+b^2=c^2=ab+ba+ca\$\$}\) will display $$a^2+b^2=c^2=ab+ba+ca$$

You can use \text{…} to type normal text in math mode. For example, $$\textrm{\$\$\sum_{i=1}^n I = \frac{n(n+1)}{2} \text{ for all } n \in \mathbb{N}\$\$}$$ will display $$\sum_{i=1}^n I = \frac{n(n+1)}{2} \text{ for all } n \in \mathbb{N}$$

Note:There are spaces before and after “for all” since math mode will ignore spacing. However, a space can be force by having a backslash before a space. For example, $a b$ will display \(ab\) while $a\ b$ will display \(a\ b\).

Formatting

  • Work can be split into sections, subsections, subsubsections, and subsubsubsections by using \section{title} or \section*{title}. The former includes a section number while the latter does not.
  • Words
    • In text mode
      • \textbf{sample
      • \textit{sample}
      • \texttt{sample}
      • \textsf{sample}
      • \underline{sample}
      • \tilde{sample}
    • In math mode
      • \mathrm{AaBbCc}
      • \mathbf{AaBbCc}
      • \mathsf{AaBbCc}
      • \mathbb{AaBbCc}
      • \mathfrak{AaBbCc}
      • \mathcal{AaBbCc}
  • Bullet Lists
    \begin{itemize}
    \item First line.
    \item Lists within lists:
    \begin{itemize}
    \item Like this.
    \item Cool!
    \item Last line.
    \end{\itemize}
