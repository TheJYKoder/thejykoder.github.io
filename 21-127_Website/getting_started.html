Getting Started

If you do want to go through the trouble of setting up a local environment for LaTeX, Overleaf is a great option for you. As Carnegie Mellon students, you can enjoy the Professional account for free, including up to 2,000 project storage and full document history.

If you do want to install LaTeX locally, you will need:

  • A text editor to create a LaTeX document (we’ll be using the freeware Texmaker)
  • A LaTeX compiler (we'll be using MiKTeX for Windows and MacTex for Mac)
  • ▶Windows (Texmaker + MikTeX)
    Downloading and installing MiKTeX for Windows
    1. Go to https://miktex.org/download
    2. After choosing “Windows,” click the Download button under “Installer”
    3. Now click Save File when the popup window opens
    4. Find where the file was saved and double click on it
    5. Windows will ask "Unknown publisher, install anyway?". Click run
    6. The Setup wizard will now open. Click the box to accept the MiKTeX copying conditions. Then click Next
    7. Click Anyone who uses this computer (all users), then click Next
    8. Feel free to change the default installation address, but you will need that address for future environment setup. After finished, click Next
    9. Feel free to change preferred paper, then click Next
    10. Click Start to begin the install process
    11. Click Close once the installation is complete
    Now you are ready to download the text editor, Texmaker for Windows
    1. Go to http://www.xm1math.net/texmaker/download.html
    2. Now click under the appropriate platform. If you are using windows, click on Executable file for win64 systems : Texmaker_5.0.3_Win_x64.msi
    3. Now click Save File when the popup window opens.
    4. Find where the file was saved and double click on it
    5. Windows will ask "Unknown publisher, install anyway?". Click run
    6. The Setup wizard will now open. Click the box I accept then click Install
    7. Click Close once the installation is complete
    Setting up Texmaker environment
    1. Open Texmaker
    2. Change the document configuration to PDFLaTeX and View PDF
    3. Under Options, open Configure Texmaker
    4. In Editor, change Editor Font Encoding to UTF-8
    5. In Commands, change Pdf Viewer to Built-in Viewer and Embed. You can also change to external viewer if you like
    6. In Commands, change the content of LaTeX to be "~/miktex/bin/x64/tex.exe"-interaction=nonstopmode %.tex
      Replace ~ in the beginning with your installation address for MiKTeX. The default value for ~ is C:/Program Files/MiKTeX 2.9
    7. In Commands, change the content of PDFLaTeX to be "~/miktex/bin/x64/tex.exe" -interaction=nonstopmode %.tex
      Replace ~ in the beginning with your installation address for MiKTeX. The default value for ~ is C:/Program Files/MiKTeX 2.9
    8. Click OK and you are all set!
  • ▶Mac (Texmaker + MacTeX)
    Downloading and installing MacTeX for Mac
    1. Go to https://tug.org/mactex/mactex-download.html
    2. Click the link called “MacTeX.pkg”
    3. Once the .pkg has been installed, just run through the installer, accepting all default settings, and you’ll be setup!
    Now you are ready to download the text editor, Texmaker for Mac
    1. Go to http://www.xm1math.net/texmaker/download.html
    2. Now click under the appropriate platform. Click on texmaker-5.0.3.dmg
    3. Once downloaded, install it like you would another application.
    Setting up Texmaker environment
    1. Open Texmaker, head to the “Preferences” screen, click on texmaker in the menu bar and then select Preferences…
    2. In the Preferences pane, under Commands, head to Pdf Viewer box and select the Embed checkbox
    3. Close the Preferences pane by clicking OK. Now click on View in the menu bar and uncheck Structure and check Pdf Viewer.
  • ▶Online (Overleaf)
    1. Go to https://www.overleaf.com/login
    2. Continue with “Log in with Google” and log in with your Andrew email
    3. After logging in, in “Account -> Account Settings”, you can verify your Andrew email and start the Professional account for free
    4. In “Projects”, click “New Project” and start to use! The LaTeX document will be automatically complied upon save
